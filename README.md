# EVNT — App de Eventos

Aplicación móvil desarrollada con Ionic + React para descubrir, explorar y gestionar eventos cercanos.

[Ver prototipo en Lovable](https://eventomon-proyect-dam.lovable.app)
[Ver repositorio en Git](https://github.com/JuanBahamon/eventomon-Proyect-DAM.git)

## Tecnologías utilizadas

- Ionic 7 + React + TypeScript
- Firebase Realtime Database
- Firebase Authentication
- Capacitor Geolocation
- Capacitor Camera
- Capacitor Motion (acelerómetro)
- React Leaflet (mapas)
- SCSS Modules

## APIs y servicios

| Servicio      |                        Uso                      |
|---------------|-------------------------------------------------|
| Firebase Auth | Login y registro de usuarios                    |
| Firebase Realtime Database | Eventos, chat en tiempo real       |
| Capacitor Geolocation | Ubicación del usuario en el mapa        |
| Capacitor Camera | Escáner QR de entradas                       |
| Capacitor Motion | Detección de agitación para resetear escáner |
| CartoDB Dark Matter | Tiles del mapa oscuro                     |
|-----------------------------------------------------------------|

## Módulos implementados

- 1) Autenticación (login / registro)
- 2) Inicio con eventos destacados y filtros
- 3) Mapa con GPS y marcadores de eventos
- 4) Buscar eventos por nombre, categoría y ciudad
- 5) Detalle de evento
- 6) Carrito de entradas
- 7) Mis entradas
- 8) Chat en tiempo real por evento
- 9) Escáner QR con cámara
- 10) Acelerómetro (shake para resetear escáner)
- 10) Perfil de usuario
